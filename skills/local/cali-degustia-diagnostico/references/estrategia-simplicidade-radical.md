# Estratégia: Simplicidade Radical

> Adaptado do arquétipo D (Radical Simplicity) do stelow — voltado para recomendações de ferramentas.

## Filosofia

Inovação por subtração. A pergunta-guia não é "qual ferramenta resolve isso?" e sim **"qual o mínimo que podemos fazer para melhorar?"**

## Quando usar como alternativa

- A tarefa tem volume baixo (<5x/semana)
- O cliente não tem orçamento ou energia para ferramenta nova
- O cliente está sobrecarregado — menos é mais
- A tarefa é determinística com regras claras
- O cliente já tem uma ferramenta que pode fazer o serviço com configuração simples

## Processo

1. Identifique a verdadeira tarefa — qual o resultado desejado, não o método atual
2. Remova interações não essenciais
3. Pergunte: **e se a gente simplesmente parar de fazer isso?**
4. Pergunte: **dá para resolver com o que já existe?** (mesma planilha, mesmo e-mail, mesmo WhatsApp)
5. Só então: **qual a ferramenta mais simples que existe para isso?**

## Exemplos

| Dor | Simplicidade Radical |
|-----|---------------------|
| "Perco 2h/dia respondendo se tem banho, se tem vaga" | Catálogo de serviços fixo na descrição do WhatsApp. Cliente lê antes de perguntar. |
| "Relatório semanal leva 3h" | Planilha modelo compartilhada com fórmulas. Time preenche durante a semana. |
| "Esqueço de follow-up com cliente" | Lembrete no calendário do Google. |
| Agendamento é vai e volta no WhatsApp | Google Agenda Booking Page, link fixo na bio. |

## Como apresentar no relatório

Sempre como **🌱 Alternativa mais simples**, em tom suave, 1 linha no card da recomendação principal. Não como seção separada.

Formato:
> 🌱 Alternativa mais simples: [descrição em 1 linha]. Setup: [tempo]. Custo: [R$ ou grátis].

## Limitações

- Não escala para volume alto
- Não resolve tarefas que exigem cognição (interpretação, nuance, síntese)
- A pessoa ainda faz a tarefa — só que de forma mais organizada

## Relação com as outras estratégias

- **Simplicidade Radical** é o ponto de partida: o mínimo que resolve
- **Padrão Convencional** adiciona IA no nível mais familiar (prompt/Custom GPT)
- **Vanguarda Tecnológica** é o máximo que a tecnologia permite (agentes, automação)
- A **Recomendação Principal** (híbrida) combina traços coerentes de cada uma
